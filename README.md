# Smart Retail Insight

## Overview

Smart Retail Insight is an AI-powered customer behavior analysis system designed for offline retail stores. By utilizing CCTV footage and computer vision technologies, the system analyzes customer movement, dwell time, and product interest to provide valuable insights for store management and marketing strategies.

## Key Features

* Customer tracking and movement analysis
* Product interest detection based on dwell time
* Real-time customer behavior monitoring
* Data visualization dashboard
* Store traffic and hotspot analysis

## Technologies

* Jetson Orin Nano
* Python
* OpenCV
* YOLO
* DeepSORT

## Expected Benefits

* Improved understanding of customer behavior
* Data-driven store layout optimization
* Enhanced marketing decision-making
* Increased operational efficiency

## Future Development

* Purchase conversion prediction
* Personalized recommendation services
* Multi-camera integrated analytics
